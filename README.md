# Burma Calendar (မြန်မာပြက္ခဒိန်)

A beautiful, modern, and high-performance Myanmar Calendar application designed to blend traditional cultural elements with state-of-the-art web technology.

## ✨ Features

- **📅 Comprehensive Calendar**: Accurate Myanmar dates and holidays for 2024, 2025, and 2026.
- **🎨 Seasonal Dark Mode**: Intelligent UI that shifts its color palette based on Myanmar's three seasons (Cool, Hot, and Rainy), with full manual Dark Mode support.
- **💎 Premium UI/UX**: Built with **shadcn/ui** and **Tailwind CSS** for a professional, eye-friendly aesthetic including glassmorphism and modern typography.
- **🎬 Fluid Animations**: Smooth transitions and micro-interactions powered by **Framer Motion**.
- **📱 PWA Ready**: Installable on desktop and mobile for a native app-like experience with offline capabilities.
- **⚡ High Performance**: Powered by **Vite** and **React** for lightning-fast load times.
- **📂 Maintainable Data**: All holiday data is centralized in a single JSON structure for easy updates.

## 🛠️ Built With

- [React 18](https://reactjs.org/) - Frontend library
- [TypeScript](https://www.typescriptlang.org/) - Type safety
- [Vite](https://vitejs.dev/) - Next-generation frontend tooling
- [shadcn/ui](https://ui.shadcn.com/) - Reusable UI components
- [Tailwind CSS](https://tailwindcss.com/) - Utility-first CSS
- [Framer Motion](https://www.framer.com/motion/) - Animation library
- [Lucide React](https://lucide.dev/) - Beautifully simple icons

## 🚀 Getting Started

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/sannlynnhtun-coding/burma-calendar.git
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Run the development server**:
    ```bash
    npm run dev
    ```
4.  **Build for production**:
    ```bash
    npm run build
    ```
